Appendix B: Glossary & References

CIA Triad
Confidentiality, Integrity, Availability — the three core security properties.
CVSS v3.1
Common Vulnerability Scoring System — industry standard for severity scoring (0.0–10.0).
ISO 27001:2022
International standard for Information Security Management Systems (ISMS).
ASD Essential 8
Australian Signals Directorate's eight essential mitigation strategies.
NIST CSF
NIST Cybersecurity Framework — Identify, Protect, Detect, Respond, Recover.
PTES
Penetration Testing Execution Standard — methodology for security assessments.
OWASP
Open Web Application Security Project — web app security standards and tools.
Nuclei
Fast, template-based vulnerability scanner with 8000+ community templates.
OWASP ZAP
Zed Attack Proxy — open-source web application security scanner.
testssl.sh
Command-line tool for thorough SSL/TLS protocol and cipher analysis.
dalfox
Open-source XSS scanner for parameter analysis and proof-of-concept generation.
SLA
Service Level Agreement — defined timeframe for remediation.
